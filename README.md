## 🚀 Getting Started

No build tools or dependencies needed — just open the file in a browser.

```bash
# Clone the repository
git clone https://github.com/adityaaldar/portfolio.git

# Navigate into the folder
cd portfolio

# Open in browser
open portfolio.html
# or on Windows:
start portfolio.html
```

## 📤 Deployment

You can deploy this portfolio for free on any of these platforms:

- **GitHub Pages** — push to a repo and enable Pages in Settings
- **Netlify** — drag and drop the `portfolio.html` file
- **Vercel** — import GitHub repo and deploy instantly

### GitHub Pages Quick Deploy

```bash
git init
git add .
git commit -m "Initial portfolio commit"
git branch -M main
git remote add origin https://github.com/adityaaldar/portfolio.git
git push -u origin main
```
Then go to **Settings → Pages → Source → main branch** and save.

## ✏️ Customization

To personalize the portfolio, open `portfolio.html` and update:

| What | Where to look |
|------|--------------|
| Name | `hero-name`, `nav-logo`, `footer` |
| Email | `contact-item` section |
| LinkedIn URL | `contact-item` anchor tag |
| GitHub URL | `contact-item` anchor tag |
| Project links | `project-links` inside each project card |
| Profile photo | `hero-photo` and `about-img-box` `<img>` src |
| Certificate images | `cert-img` `<img>` src |
